DeepSEA data

Download the data for training the DCNN:

```
wget -O DeepSEA_data.tar.gz https://cloud.tsinghua.edu.cn/d/fee522536d524eae9531/files/?p=%2FDCNN_train%2FDeepSEA_data.tar.gz&dl=1

tar -xzvf DeepSEA_data.tar.gz
mv data/* ./
```
